# 第 4 回


授業中にお伝えしました 5 月 5 日分の課題授業に関してメールいたします。

* 実際の論文は，授業の GitHub ページからリンクがありますが，以下にも URL を記載します: https://komazawa-deep-learning.github.io/2023/2020Lindsay_attention_ja.pdf
