## ロジスティック回帰 (Logistic regression)

$$
P(x) = \left[1+e^{-x}\right]^{-1} =\frac{1}{1+e^{-x}}
$$

もう少し一般化すると項目反応理論に用いられる回帰式 $\displaystyle P(x)=\frac{1}{1+e^{-a \left(x-b\right)+c}}$$ となる。

```python
import matplotlib.pyplot as plt
import numpy as np
try:
```
